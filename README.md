English

Project Overview
This plugin is designed for automated activity tracking and data logging. It monitors player statistics, stores them in a database, and ensures administrators stay informed through real-time notifications.

Key Features
Data Logging: Automatically tracks mining statistics and stores all data in a centralized database for long-term storage.
Instant Notifications: Sends regular updates and reports directly to Discord and Telegram.
Smart Alerts: If a player mines more than 45 diamonds, the plugin will immediately send a Direct Message (DM) to the administrator on Telegram and Discord to prevent potential abuse.
Continuous Improvement: This plugin is currently in active development. It will be further improved and updated with new features and optimizations in the future.

Русский

Описание проекта
Этот плагин предназначен для автоматического отслеживания активности и логирования данных. Он мониторит статистику игроков, сохраняет её в базу данных и информирует администраторов через уведомления в реальном времени.

Основные функции
Логирование данных: Автоматически отслеживает статистику добычи ресурсов и сохраняет всю информацию в централизованную базу данных.
Мгновенные уведомления: Отправляет регулярные отчеты и обновления прямо в Discord и Telegram.
Умные алерты: Если игрок добывает более 45 алмазов, плагин мгновенно отправляет личное сообщение (ЛС) администратору в Telegram и Discord для предотвращения возможных злоупотреблений.
Постоянное развитие: Плагин находится в активной разработке. В будущем он будет совершенствоваться, дополняться новыми функциями и оптимизироваться.
